# PROMUEVA DATA

This repository contains a collection of anonymized datasets gathered from X (formerly Twitter) and YouTube as part of the [PROMUEVA Project](https://sites.google.com/view/promueva/english-version). The datasets were curated and processed to support research on opinion dynamics, and polarization.

Each dataset includes user comments and interaction data related to specific topics. To ensure privacy, all datasets have been fully anonymized and are shared for research and educational purposes.

If you need more details about the datasets or the PROMUEVA project, please contact the PROMUEVA Team at: promueva@googlegroups.com
